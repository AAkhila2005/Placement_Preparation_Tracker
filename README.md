# 🎯 Placement Preparation Tracker

An interactive web application built using **HTML, CSS, Bootstrap, JavaScript, and Chart.js** to help students track their placement preparation progress. The project includes CRUD operations, data persistence using Local Storage, progress analytics, search functionality, and CSV export.

---

## 📌 Overview

This project is designed to help students monitor and organize their placement preparation by tracking progress across **DSA, Aptitude, and Core Subjects**. It provides a simple and responsive interface with real-time analytics to visualize preparation status.

---

## 📂 Features

- Add placement preparation records
- Edit existing records
- Delete records
- Search records by topic
- Dashboard with live progress statistics
- Interactive Pie Chart visualization
- Export progress records as CSV
- Persistent data storage using Local Storage
- Responsive design for desktop and mobile

## 📈 Project Workflow

### 1. Data Entry
- Enter placement preparation topics.
- Select category (DSA, Aptitude, Core Subject).
- Record the number of problems or topics completed.

### 2. Data Management
- Add new progress records.
- Edit existing records.
- Delete unnecessary records.
- Automatically save data using Local Storage.

### 3. Dashboard Analytics
- Display total DSA problems solved.
- Display Aptitude progress.
- Display Core Subject progress.
- Update analytics automatically after every change.

### 4. Data Visualization
- Generate an interactive Pie Chart using Chart.js.
- Visualize preparation distribution across categories.

### 5. Search & Export
- Search topics instantly.
- Export all records as a CSV file for future reference.
